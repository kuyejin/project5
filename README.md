# 📂 프로젝트 소개 
​
안녕하세요! <br/>
**T셀파몰**은 교사 대상 온라인 중고거래 사이트는 교사들이 <br/>
중고 제품을 거래하고 판매하는 데 특화된 플랫폼입니다.
​<br/><br/>
교사들은 자신이 사용하지 않는 교육 자료, 교과서 등을 <br/>
T셀파몰 사이트에서 판매하거나 구매할 수 있습니다.  <br/>
이러한 교사 대상 중고거래 사이트는 교육 커뮤니티를 지원하며 <br/>
**교사들 간의 자원 공유와 경제적 이익을 증진시키는 데 기여** 합니다
